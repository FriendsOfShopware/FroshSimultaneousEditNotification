Das Plugin zeigt im Backend bei Bestellungen, Kunden, Artikeln und Einkaufswelten an, ob gerade mehrere User das selbe Element bearbeiten. Für das Plugin wird ein Google Firebase Projekt benötigt

Firebase Anleitung

  • Erstelle unter https://console.firebase.google.com/ eine neue Anwendung und kopiere das Javascript Setup Script in die Plugin Konfiguration
  • Setze die Berechtigungen für die Realtime Datenbank https://firebase.google.com/docs/database/security/quickstart
  • Gehe dazu auf dein Firebase Projekt (Links in der Sidebar) auf Database
  • Erstelle hier eine Datenbank in test-mode

Die Datenbank Rechte sollten folgender maßen sein 

{  "rules": {
      ".read": true,
      ".write": true
      }
    }

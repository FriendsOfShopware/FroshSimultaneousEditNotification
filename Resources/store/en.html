The plugin shows in the backend for orders, customers, articles and shopping worlds whether several users are currently editing the same element. A Google Firebase project is required for this plugin.

Firebase Setup Guide

  • Create new application on https://console.firebase.google.com/ and put js setup code in the plugin configuration
  • set public permission rule for the Database https://firebase.google.com/docs/database/security/quickstart

The database right configuration should be 

{  "rules": {
      ".read": true,
      ".write": true
      }
    }
